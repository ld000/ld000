# Profile Image Credits

`hero-room.png` was generated with GPT Image 2 from an original prompt for this
profile, then resized locally for the README banner.

The supporting PNG images are generated locally by
`scripts/update_pixel_profile.py`. The script uses only Python standard library
modules. It draws pixel-art panels, portal buttons, route-map props, project
cards, icons, and a tiny built-in bitmap font, then writes PNG files directly
through a small zlib-based PNG writer.

No hotlinked images, external badge services, or third-party copyrighted PNG
assets are used in the README visuals.
