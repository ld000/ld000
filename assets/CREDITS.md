# Profile Image Credits

The current README images are two locally generated PNG files:

- `assets/hero-signal-passport.png`
- `assets/signal-panel.png`

Both are generated by `scripts/update_pixel_profile.py` using only Python
standard library modules and public GitHub API data. The script draws the Daily
Stamp / Signal Passport visuals directly and writes PNG files through a small
zlib-based PNG writer.

No hotlinked images, external badge or card services, or third-party
copyrighted PNG assets are used. Older pixel-room, toy-packaging, build-mode,
portal, and project-card directions are retired.
