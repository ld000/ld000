# Profile Image Credits

The README PNG images are generated locally by
`scripts/update_pixel_profile.py`. The script uses only Python standard library
modules. It draws toy-packaging and arcade-panel assets, stickers, project
cards, icons, halftone texture, and a tiny built-in bitmap font, then writes PNG
files directly through a small zlib-based PNG writer.

Legacy pixel-room assets from the previous README version remain in `assets/`
for rollback, but are no longer referenced by the README.

No hotlinked images, external badge services, or third-party copyrighted PNG
assets are used in the README visuals.
